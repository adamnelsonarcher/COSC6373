Homework 06 – ICA – Part A 
Binary Classification Metrics 
Overview 
In this homework, we aim to evaluate the performance of a binary classification model. 
Description 
We want to use transfer learning with a pre-trained ResNet50 CNN model trained on the ImageNet 
dataset [1] (spanning 1,000 object  classes and containing 1,281,167 training  images) to classify 
images of horses and camels using the dataset provided in [2]. More specifically, fine-tuning refers 
to the practice of "unfreezing" a few of the top layers of a frozen model base and jointly training 
both  the  newly  added  classifier  layers  and  the  base  model's  last  layers.  This  allows  us  to  "fine-
tune" the higher-order feature representations in the base model to make them more relevant for 
the specific task that we retrain our model to perform. Note: the link to the notebook on training a 
CNN from scratch with Keras is in the Reading sheet (R6).
