
HW 07 ICA Part A  
Building a multimodal AI Chatbot Agent with Intent Understanding, Memory, Web 
Search, and Visual Analysis 
In  this  assignment,  you  will  design  and  implement  a  Python-based  AI  chatbot  that  uses  open-weight 
language and vision models through Ollama and LangChain. 
Your chatbot should: 
● Use Mistral (LLM) for conversation and reasoning 
● Use LLaVA (VLM) for image understanding 
● Detect user intent before performing any action 
● Maintain short-term conversational memory (within a single chat session) and persistent long-
term memory across sessions using text embeddings 
● Know the current time and date through a hidden system prompt 
● Search the web when information is not available in memory 
● Handle camera access and perform image analysis using the VLM 
Part A — Models, Tools, and Intent Understanding 
A1. Models and Platform 
● Use Ollama to run open-weight models locally: 
○ Mistral — Large Language Model (LLM) for text-based reasoning and dialogue 
○ LLaVA — Vision-Language Model (VLM) for image interpretation and analysis 
● Use LangChain to control Ollama through Python. 
● Users must be able to chat with the chatbot directly through the system command line, 
entering text prompts and viewing responses in the console.

Additional Requirements 
● The chatbot must determine intent before calling any tool or function. 
● For each user message, a hidden system prompt must be appended that includes the current 
date and time, invisible to the user but available to the LLM for reasoning. 
● A policy table must map each detected intent to the appropriate system behavior or tool (to be 
implemented in Part B). 
A3. Evaluation 
● Test the intent classifier on a small labeled dataset (20 examples). 
● Provide summary metrics such as intent classification accuracy and include a confusion matrix. 
● Assume that user input prompts are single intent only
